# Flipkart Water Purifiers Web Scraping and Data Analysis using Python

## 📖 Project Overview

This project extracts water purifier product data from Flipkart using Python and BeautifulSoup. The collected data is cleaned, validated, and analyzed using Pandas and NumPy. Data visualization is performed using Matplotlib to identify pricing trends, customer ratings, and product insights.

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- BeautifulSoup
- Requests
- Matplotlib
- Jupyter Notebook

---

## 📂 Dataset

The dataset contains information collected from Flipkart, including:

- Product Name
- Brand
- Price
- Customer Rating
- Number of Reviews

## 📊 Project Workflow

1. Web Scraping using BeautifulSoup

3. Exploratory Data Analysis (EDA)
